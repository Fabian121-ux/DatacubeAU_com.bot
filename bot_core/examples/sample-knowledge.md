# Datacube AU Overview

Datacube AU is building a production-grade WhatsApp AI bot platform with WAHA Core, FastAPI, and PostgreSQL.

## Group Behavior

In groups, the bot should operate in mention-only mode by default to avoid spam.

## DM Behavior

In direct messages, the bot can answer static questions and knowledge-backed questions.

## Local Testing

Use the admin knowledge endpoints to upload or paste text, then query the webhook with a message that matches the document content.
